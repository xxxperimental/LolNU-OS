List of commands:

execute <program_name> - Executes the program.
type <file_name> - Prints out the contents of the file.
delete <file_name> - Deletes the file.
copy <file_name> <file_name> - Copies the contents of the first file into the second file and creates it.
dir - Lists the contents of the directory. 
create <file_name> - Creates text file with the given name. Accepts lines of input until empty line.
clear - clear the console.
kill - kill the current process.
help - view the help menu.
color-help - view the color list.
